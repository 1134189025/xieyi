## 敏感信息脱敏验证

已覆盖：

| 路径 | 结果 |
| --- | --- |
| 外包提交接口业务失败 | `activationCode` 和 Pix 码会写成 `[redacted-activation-code]`、`[redacted-pix-code]` |
| 外包提交接口 HTTP 非 2xx | 上游响应体里的外包码和 Pix 码会脱敏后进入诊断 |
| 外包兑换码 `code-info` 业务失败 | 上游 message 里的明文外包码会脱敏后保存到 `lastError` |
| 外包兑换码 `code-info` HTTP 非 2xx | HTTP 错误响应里的明文外包码会脱敏后保存到 `lastError` |
| 外包订单终态轮询失败 | 上游 `last_error` 会结合订单关联外包码上下文和 Pix 正则脱敏后写入订单诊断 |
| 外包状态批量查询异常 | `/orders-status` 整体 HTTP/JSON 异常会按本批订单外包码上下文脱敏，维护循环日志只记录安全结构化字段 |
